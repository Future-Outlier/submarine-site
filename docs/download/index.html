The latest release of Apache Submarine is 0.7.0.

  • Apache Submarine 0.7.0 released on April 25, 2022 (release notes) (git tag)

    • Binary package: submarine-dist-0.7.0.tar.gz (148 MB, checksum, signature)

Verify the integrity of the files‚Äã

It is essential that you verify the integrity of the downloaded files using the PGP or MD5 signatures. This signature should be matched against the KEYS file.

gpg --import KEYS
gpg --verify submarine-dist-X.Y.Z-src.tar.gz.asc
